# Criando relatorios dinamicos com o uso de parametros no Power BI

Sobre o projeto
----

O projeto consiste na utilização do relatório financeiro criado na aula anterior com o objetivo de inserir parâmetro e tornar o relatório dinâmico.
O parâmetro utilizado foi inserido na pagina de detalhes de vendas do relatório, sendo apresentado os elementos Segmento, Trimestre, Proft e Produt.
